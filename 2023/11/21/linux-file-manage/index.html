这里展示一些linux系统下常用的文件操作命令，用来管理文件和系统目录。

1. Linux的目录结构

Linux的目录结构是层次化的树状结构，文件目录的顶层即为根目录/，所有文件和目录都是根目录的衍生，以子文件或者是子目录的形式被倒挂在根目录之下，构成整个linux文件系统。

   graph TD;
    / --> bin
    / --> boot
    / --> dev
    / --> etc
    / --> 1[...]
    bin --> apt
    bin --> vim
    bin --> 2[...]
    etc --> init.d
    etc --> 3[...]
    boot --> 4[...]
    dev --> 5[...]
    1[...] --> 6[...]
