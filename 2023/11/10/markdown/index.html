Markdown的常见语法

2023/11/10

测试

markdown是一个轻量级的标记语言，它可以基于纯文本的形式来对文档进行编辑和修改。 这里展示一些markdown的常见语法，方便个人在hexo博客下撰写自己的文章。

1. 标题

如果要在文章中创建标题，应该在标题文字前面添加上对应标题级别的井号#，井号数量从一个到六个分别对应一级到六级标题。比如要创建一个四级标题，则对应的markdown写法为：

#### Heading level 4
pnpm add hexo-babel
#home-card #card-div .avatar {
    border: #f1f1f1 solid 3px;
    border-radius: 50%;
    height: 140px;
    margin: auto;
    text-align: center;
    width: 140px;
}
